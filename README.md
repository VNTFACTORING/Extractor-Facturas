# VanTrust Capital · Extractor de Facturas

Herramienta interna para extraer datos de facturas PDF e imágenes usando IA.

## Deploy en Vercel (5 minutos)

1. Sube este repositorio a GitHub
2. Entra a [vercel.com](https://vercel.com) y conecta tu cuenta de GitHub
3. Importa este repositorio
4. En **Environment Variables**, agrega:
   - `ANTHROPIC_API_KEY` → tu API key de Anthropic
5. Haz clic en **Deploy**

La URL resultante (ej: `extractor-vantrust.vercel.app`) es la que compartes con los ejecutivos.

## Uso

1. Arrastra una o varias facturas (PDF o imagen)
2. Haz clic en **Extraer datos**
3. Copia el resultado con **Exportar a Excel / GVE**
